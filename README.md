# Gustavo Graciano

Engenheiro de software full-stack — construo **APIs em .NET** e **frontends em React/TypeScript** de ponta a ponta, com foco em sistemas multi-tenant, segurança (JWT + RBAC), auditoria e qualidade (testes + CI/CD).

```text
bash ~ $ whoami
→ Backend: .NET 8/9 · EF Core · PostgreSQL/SQL Server · JWT · xUnit · Clean Architecture
→ Frontend: React 18/19 · TypeScript · MUI · Vite · Zustand · React Router
→ Mobile: React Native · Expo · SQLite offline
→ DevOps: Docker · Azure App Service · Azure Static Web Apps · GitHub Actions
```
